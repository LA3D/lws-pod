# superpowers — build history (archive)

These `specs/`, `plans/`, and `findings/` are **append-only records of how each rung was built** —
the brainstorm → spec → plan → implement trail for each workstream. They are history, **not current
truth**, and some links/decisions in them are superseded.

Do **not** mine these for the current design. Instead:

- **Current state + open items** → [`../../FOLLOWUP.md`](../../FOLLOWUP.md)
- **Forward plan / order of operations** → [`../ROADMAP.md`](../ROADMAP.md)
- **Settled canon** → [`../foundations/`](../foundations/)
- **Active design deliberation** (exploratory) → [`../design-notes/`](../design-notes/)

Read a spec/plan here only when you specifically need *how a past rung was built and why*.
